# Software Requirements Specification (SRS) (& System testing)

## Project title
Code Gardener

## Project description
Code Gardener는 개발자가 작성한 완성된 코드 결과물(개인 프로젝트, 코딩테스트 풀이 등)에 대한 피드백을 제공하는 플랫폼이다. 많은 개발 학습자들이 결과물에 대해 체계적인 피드백을 받을 환경이 부족하다는 문제를 인식하고 이를 해결하고자 기획되었다. 사용자가 코드를 업로드하면 AI가 즉각적인 피드백을 제공하고, 다른 사용자들이 가독성, 효율성, 문제 해결 방식 등 다양한 관점에서 의견을 더한다. 이 과정에서 사용자는 서로의 코드를 돌보는 가드너(Gardener) 로서 참여하게 되며, 이를 통해 역량 개선, 지식 공유와 성장의 기회를 얻는다. 본 서비스는 건강한 코드 리뷰 문화를 확산시키고, 개발자들이 함께 성장할 수 있는 환경을 제공하는 것을 목표로 한다.
